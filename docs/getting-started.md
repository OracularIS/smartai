# Smart AI – Getting Started

Smart AI allows you to interact with enterprise systems securely and intuitively. Using the **Secure Chat** feature, you can:

- Chat with the system using natural language  
- Retrieve data and insights across multiple enterprise systems  
- Generate visualizations like **pie charts**, **bar charts**, and **line charts**  
- Inspect the **underlying function names and code** behind each operation  
- View results directly on your **dashboard**  

All operations are executed safely within enterprise boundaries, using only pre-approved functions.

---

## Step 1: Access Smart Functions Portal

Go to [Smart Functions Portal](https://smartfunctions.smart-is.com/).  
The landing page appears as shown below:

![Smart FX Login Page](.attachments/01_smartfx_login_page.png)  
Click **Get Started Free** or **Sign In**.

---

### Step 2: Login Options

The login window allows you to sign in using:

- Email & Password  
- Microsoft SSO  
- Google SSO  

![SSO Login Page](.attachments/02-sso_login_page.png)

---

### Step 3: Add Your Project

After logging in, the **Add Your Project** window appears.  
Here you can enter:

- **Project Name**  
- **Repository Link**  
- **Personal Access Token (PAT)**  

Then click **Clone & Create Project** to proceed.

![Add Your Project](.attachments/03_add-your_project.png)

## Example Use Case from Video

1. Go to **Secure Chat** and select a system (e.g., Enterprise)  
2. Click **Connect** → Message appears: `connected_successfully`  
3. Enter queries in the chat box:  
   - `"Show me user information across the enterprise"` → returns **Tables, Insights, and Code**  
   - `"Make a pie chart for users across all systems"` → returns **Chart and Code**  

You can then interact further with the results or add them to your dashboard.

---

**Notes:**

- All operations are executed within secure enterprise boundaries  
- No unauthorized or ad-hoc system access is permitted  
- Smart App Keys should be stored securely and never shared publicly  

For more information, visit [Smart Functions Portal](https://smartfunctions.smart-is.com/).
