# Smart AI – Getting Started Guide

Smart AI enables secure and intuitive interaction with enterprise systems through natural language. Using the **Secure Chat** feature, you can:

- Chat with the system using natural language  
- Retrieve data and insights across multiple enterprise systems  
- Generate visualizations such as pie charts, bar charts, and line charts  
- Inspect the underlying function names and execution code  
- View and manage results directly on your dashboard  

All operations are executed securely within enterprise boundaries using only pre-approved functions.

---

## Step 2: Login Options

You can sign in using one of the following methods:

- Email and Password  
- Microsoft SSO  
- Google SSO  

![SSO Login Page](.attachments/02-sso_login_page.png)

## Step 3: Add Your Project

After successful login, the **Add Your Project** screen will appear.

Provide the following details:

- Project Name  
- Repository Link  
- Personal Access Token (PAT)  

Click **Clone & Create Project** to proceed.

![Add Your Project](.attachments/03_add-your_project.png)

### Developer Options

The right-side panel provides additional capabilities:

- View and modify the underlying function code  
- Execute functions directly from the Dev Console  
- Pull and push changes to the connected branch  

You can also change the connector associated with a function from this panel.

## Example Use Case

1. Navigate to **Secure Chat** and select a system (for example, Enterprise)  
2. Click **Connect** and confirm the `connected_successfully` message  
3. Enter queries such as:

- "Show me user information across the enterprise"  
  Returns tables, insights, and underlying code  

- "Make a pie chart for users across all systems"  
  Returns a chart along with the corresponding code  

You can further interact with the results or add them to your dashboard.

---

## Notes

- All operations are executed within secure enterprise boundaries  
- Only pre-approved functions are allowed to run  
- Unauthorized or ad-hoc system access is not permitted  
- Smart App Keys must be stored securely and should never be shared publicly
